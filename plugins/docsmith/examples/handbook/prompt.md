# Example prompt

A user would type something like this to the `/make-pdf` skill to produce this PDF:

```
Turn async-handbook.md into a handbook PDF for Acme Corp.
```

This demo showcases the full `handbook` template as a visual component catalog:
every callout card (note, tip, warning, do, dont, plain, cheatsheet, pullquote),
the automatic chapter openers and dotted-leader table of contents, clickable
external links, a hand-written raw SVG process diagram, and a two-column glossary
table — all rendered into a navy-branded LaTeX book.
