# Example prompt

```
Make a cute kawaii-storybook PDF from cloud-safety.md, branded as Nimbus Studio.
```

This renders the `kawaii-storybook` template end to end — a soft pastel 16:9 deck
that doubles as a visual catalog of every signature layout: cover, bigstat,
pillars, the three colour-coded `path` verdict pills (accept / reject / caution),
laws, scorecard, flow, scenarios, roadmap, a hand-written SVG `figure` hero,
a fenced code card, `tip` and `warning` callouts, quote, statement, and closing.
