# Example prompt

```
Make a corporate-deck PDF from acme-qbr.md, brand it as Acme Corp.
```

This renders the `acme-qbr.md` source into a 16:9 boardroom deck and doubles as a
visual catalog of every `corporate-deck` slide class — cover, agenda, bigstat,
kpi, pillars, cards3, compare2, split, steps, timeline, versus, procon,
iconcards, stack, quote, statement, and closing — with hand-written raw SVG for
the architecture, roadmap, and icon-card figures.
